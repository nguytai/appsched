How to setup the site?

Step 1: Install Composer (https://getcomposer.org/download/)
<br />
Step 2: Delete composer.lock file if it exists (rm composer.lock)
<br />
Step 3: Install Vendor files (composer install)
<br />
Step 4: Update Vendor files (composer update)
<br />
Step 5: Install MAMP (https://www.mamp.info/en/downloads/)
<br />
Step 6: Visit http://localhost:80
